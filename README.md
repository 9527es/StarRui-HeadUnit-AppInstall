- 笔记本电脑（推荐Windows 10/11系统，兼容性更佳）

- 手机（开启热点，用于车机联网）
- U盘（FAT32格式，8G及以上，仅降级方案需用）

3. 前置检查
 
1. 记录车机信息：设置 → 系统信息 → 关于本机，记录版本号、硬件标识（如ihu353）、MCU版本（后续适配方案关键依据）。
2. 备份数据：导航记录、蓝牙配对信息、车机内重要文件（避免操作异常导致数据丢失）。
3. 车辆状态：车辆熄火通电（不启动发动机），关闭车机省电模式，确保车机电量充足（建议连接车辆充电器）。
 
📝 分场景操作教程
 
场景一：免降级ADB激活法（2024年前车型/版本h≤1346）
 
步骤1：车机联网与基础设置
 
1. 手机开启热点，车机连接该热点（设置 → 网络 → WLAN，输入热点密码，确保网络稳定）。
2. 车机打开拨号键盘，输入指令  #*32280 ，自动跳转至安卓原生设置界面。
3. 滑动找到「安全」选项，勾选「允许安装未知来源应用」（部分版本显示为「允许来自此来源的应用」）。
 
步骤2：激活开发者模式
 
1. 返回设置主界面，找到「关于本机」，连续点击「版本号」7次，直至弹出「已进入开发者模式」提示（部分车型需点击「软件版本」）。
2. 返回设置界面，新增「开发者选项」，点击进入后：
- 勾选「USB调试」
- 勾选「对SD卡进行读写保护」（避免安装时权限不足）
 
步骤3：电脑与车机连接
 
1. 用双向USB线连接电脑USB口与车机USB接口（优先选择主驾驶位USB口，部分车型副驾接口仅支持充电）。
2. 电脑安装并打开应用宝PC版，等待软件自动识别车机设备（识别成功后，左上角会显示车机名称，如「Geely StarRui」）。
- 若未识别：更换USB线，或在设备管理器中更新安卓设备驱动（应用宝会自动安装驱动，若失败可手动安装）。
 
步骤4：安装第三方应用
 
1. 电脑端应用宝搜索目标车载应用（如「高德地图车机版」），点击「安装到车机」，等待自动推送安装。
2. 手动安装方式：将下载好的APK安装包拷贝至U盘，插入车机 → 应用宝PC版 → 我的手机 → 文件管理 → 导入U盘文件 → 找到APK文件，点击「安装」。
3. 安装完成后，车机桌面会显示应用图标，直接打开即可使用。
 
场景二：工程模式激活法（2024款h>1346版本）
 
步骤1：进入工程模式与开发者选项
 
1. 车机拨号键盘输入指令  #*32280 ，进入安卓原生设置界面。
2. 滑动至底部「关于智能平台」，连续点击「版本号」5次，激活开发者选项（弹出提示即成功）。
3. 返回设置 → 开发者选项，开启「WIFI Adb debugging」（WIFI ADB调试），记录界面显示的车机IP地址（如「172.20.10.4」，后续连接需用）。
4. 关闭「USB调试」后重新开启，确保调试权限生效。
 
步骤2：星瑞助手连接车机
 
1. 电脑解压「星瑞助手APP压缩包」，双击运行.exe程序（无需安装）。
2. 在程序界面输入记录的车机IP地址，默认端口为5555，点击「连接」。
- 连接失败：检查车机与电脑是否在同一热点网络，重启车机后重新尝试。
3. 连接成功后，点击「安装应用」按钮，选择本地下载好的车载APK安装包，软件会自动推送至车机并完成安装。
 
步骤3：后续配置
 
1. 安装完成后，关闭车机「WIFI Adb debugging」（避免占用网络资源，降低安全风险）。
2. 长按车机屏幕左上角电源键，选择「重启」，确保应用正常运行。
 
场景三：免拆破解法（2024款ihu353标识车型）
 
步骤1：激活车机隐藏设置入口
 
1. 车机连接手机热点，打开车机自带输入法 → 左上角「小键盘图标」→ 更多设置 → 使用帮助 → 隐私政策。
2. 下拉隐私政策页面，找到蓝色字体「账号与安全」，长按该文字3秒后选择「打开链接」。
3. 同时关闭手机数据网络（仅保留热点连接），待弹出「网络不可用」提示时，快速点击返回键，同时点击屏幕右上角隐藏的「设置」入口（需快速操作，多尝试几次）。
 
步骤2：配置开发者模式
 
1. 进入隐藏设置界面后，找到「关于智能平台」→ 状态信息，记录车机IP地址。
2. 连续点击「版本号」7次，激活开发者选项，返回后开启「USB调试」和「WIFI ADB」。
 
步骤3：电脑连接与应用安装
 
1. 用双向USB线连接电脑与车机，打开应用宝PC版，等待设备识别成功。
2. 通过应用宝安装「车载应用管理器」（搜索即可），后续可直接通过该工具管理和安装第三方应用，无需重复激活。
 
场景四：系统降级法（2023年10月后无WIFI ADB车型）
 
⚠️ 警告：此方案仅适用于无其他激活路径的车型，系统降级有一定风险（如操作不当可能导致车机无法开机），建议优先尝试前三种方案，操作前务必备份数据！
 
步骤1：下载并准备降级包
 
1. 通过「吉利车机资源站」下载对应车型的低版本系统包（需支持WIFI ADB功能，版本号建议选择h≤1346）。
2. 将降级包解压至U盘根目录（确保解压后文件直接在根目录，而非子文件夹），U盘格式化为FAT32格式。
 
步骤2：车机降级操作
 
1. 将U盘插入车机USB接口，车机拨号键盘输入指令  #*32281 ，进入后台系统更新界面。
2. 选择「系统更新 → 本地更新」，软件会自动扫描U盘中的降级包，点击「确认更新」。
3. 等待更新完成（约15-20分钟，期间车机会自动重启多次，严禁熄火、拔U盘或断电）。
4. 更新完成后，车机自动重启，进入低版本系统。
 
步骤3：激活ADB与安装应用
 
1. 重启后，参考「场景一：免降级ADB激活法」的步骤1-4，激活开发者模式、连接应用宝，即可安装第三方应用。
 
⚠️ 避坑要点与风险控制
 
1. 版本适配禁忌
 
- 严禁安装手机版应用（非车载版应用可能导致画面变形、触摸失灵、车机卡顿，甚至影响核心功能）。
- 2024款h>1346版本车机，建议安装不超过3个第三方应用，避免占用过多内存导致语音控制、导航等原生功能失效。
- 安装包需选择适配1920*720分辨率的车载版（星瑞车机默认分辨率），否则会出现显示异常。
 
2. 操作注意事项
 
- 所有操作需在车辆熄火通电状态下进行，避免发动机启动时电压波动影响车机。
- 系统降级过程中，若出现卡住、黑屏等情况，不要强制断电，等待30分钟后仍无反应，再联系吉利4S店处理。
- 激活后及时关闭「WIFI ADB调试」，仅在安装应用时开启，防止网络安全风险。
 
3. 常见故障处理
 
故障现象 排查方案 
应用宝无法识别车机 1. 更换支持数据传输的USB线；2. 重启车机与电脑；3. 手动更新安卓设备驱动 
安装应用时提示「权限不足」 1. 确认开发者选项中「USB调试」已开启；2. 重新勾选「允许安装未知来源应用」；3. 重启车机后重试 
安装后车机卡顿、反应变慢 1. 卸载不常用应用；2. 通过应用宝清理车机缓存；3. 恢复出厂设置后重新激活 
车机原生功能（如语音控制）失效 1. 卸载最近安装的应用（可能存在兼容冲突）；2. 关闭开发者模式中的冗余选项
